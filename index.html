Env Dependencies

In order to use this library it needs to have access to your coinkite secret and public key. There are a few unittests in place to verify that you have set them correctly. In general you will want to pass them as jvm runtime arguments.

java -jar <yourjarname> -DX-CK-Key=<your key> -DX-CK-Sign=<your secret key>
